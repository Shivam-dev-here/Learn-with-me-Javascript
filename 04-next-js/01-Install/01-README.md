# Installation and Setup

#### Terminal : Offline

- Nodejs required 
-  `npx create-next-app@latest`
	1. Yes: TypeScript
	2. No: Eslint
	3. Yes: Tailwind CSS
	4. No: src/ directory
	5. Yes: App Router
	6. No: Import alias
- `npm run dev`
