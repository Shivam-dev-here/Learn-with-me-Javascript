#### DOM and BOM
- Document Object Model = Entier mapping of our content in a object
- Browser Object Model = Dom + Extra
### Use

- manipulate it
- making other element depedndent as behaviour
### Tricks
- inspect > select any element > Use ''' $0 ''' in console > Manipulate
