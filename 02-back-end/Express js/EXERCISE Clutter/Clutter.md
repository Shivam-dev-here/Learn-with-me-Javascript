# What is Cluttering

- Clutter = Grouping in proper category for different different randomly shattered files

- Example: Shattered Files

```
Root
├──  img1.jpeg
├──  vid1.mp4
├──  text.pdf
├──  img2.jpeg
│
.
.
.
└── Clutter
```

- Example: Cluttered

```
Root
├── img/ img1.jpeg, img2.jpeg
├── video/ vid1.mp4
├── text/  text.pdf
|
.
└── Clutter
```
